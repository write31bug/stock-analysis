# 股票技术分析系统 - 本地使用指南

> 基于 stock-analysis v1.8.0 | 适用于 macOS / Linux / Windows
>
> 包含 CLI 命令行工具 + Web 可视化界面

---

## 一、环境准备

### 1.1 系统要求

- Python >= 3.8
- Node.js >= 18（仅 Web 界面需要）
- pip（Python 包管理器）
- 网络连接（实时数据需要，离线测试不需要）

### 1.2 获取项目

```bash
# 假设项目已克隆到本地
cd stock-analysis
```

### 1.3 安装

```bash
# 方式一：仅 CLI 工具（最小安装）
pip install -e .

# 方式二：CLI + 全部数据源
pip install -e ".[full]"

# 方式三：CLI + Web 可视化 + 全部数据源（推荐）
pip install -e ".[full,web]"

# 方式四：含开发测试工具
pip install -e ".[full,web,dev]"
```

### 1.4 验证安装

```bash
stock-analyze --version
# 输出: stock-analyze v1.8.0
```

## 二、快速上手

### 2.1 离线测试（无需网络）

```bash
# 分析单只股票（模拟数据）
stock-analyze 600519 --test

# 分析港股
stock-analyze 00700 -m hkstock --test

# 分析美股
stock-analyze AAPL -m usstock --test

# 分析 ETF 基金
stock-analyze 159934 -t fund --test

# 分析开放式基金
stock-analyze 001316 -t fund --test
```

### 2.2 实时分析（需要网络）

```bash
# A 股（自动识别沪深）
stock-analyze 600519          # 贵州茅台
stock-analyze 000001          # 平安银行

# 港股
stock-analyze 00700 -m hkstock   # 腾讯控股

# 美股
stock-analyze AAPL -m usstock    # 苹果
stock-analyze TSLA -m usstock    # 特斯拉

# ETF 基金
stock-analyze 510310 -t fund     # 沪深300ETF
stock-analyze 159934 -t fund     # 黄金ETF

# 开放式基金（需显式指定 -t fund）
stock-analyze 001316 -t fund     # 安信稳健增值
```

## 三、命令行参数速查

### 3.1 完整参数列表

| 参数 | 缩写 | 说明 | 示例 |
|------|------|------|------|
| `stock_code` | - | 股票/基金代码（位置参数） | `600519` |
| `--market` | `-m` | 市场：`auto` / `ashare` / `hkstock` / `usstock` | `-m hkstock` |
| `--type` | `-t` | 资产类型：`stock` / `fund` | `-t fund` |
| `--days` | `-d` | 历史数据天数（默认 60） | `-d 120` |
| `--batch` | `-b` | 批量分析（逗号分隔） | `-b 600036,600900` |
| `--json` | `-j` | JSON 输出 | `--json` |
| `--pretty` | `-p` | 格式化 JSON（需配合 --json） | `--json --pretty` |
| `--test` | - | 离线测试模式 | `--test` |
| `--ascii` | - | ASCII 模式（中文替换为英文） | `--ascii` |
| `--output` | `-o` | 输出到文件 | `-o result.json` |
| `--check` | - | 环境自检 | `--check` |
| `--verbose` | - | 详细日志输出 | `--verbose` |
| `--quiet` | - | 静默模式 | `--quiet` |
| `--watchlist` | `-w` | 使用持仓列表分析 | `--watchlist` |
| `--add` | - | 添加到持仓列表 | `--add 600519` |
| `--remove` | - | 从持仓列表删除 | `--remove 600519` |
| `--list` | - | 查看持仓列表 | `--list` |
| `--version` | `-v` | 显示版本号 | `--version` |

### 3.2 代码自动识别规则

| 代码特征 | 自动识别为 |
|---------|-----------|
| 6 位数字，6/5/7/9 开头 | A 股（上海） |
| 6 位数字，其他开头 | A 股（深圳） |
| 6 位数字，15/51/58 开头 | ETF 基金 |
| 6 位数字，16 开头 | LOF 基金 |
| 含 `HK.` 前缀 | 港股 |
| 纯字母 | 美股 |

> 注意：00/01 开头的开放式基金与股票代码重叠，**必须**加 `-t fund` 指定。

## 四、批量分析

### 4.1 批量股票

```bash
stock-analyze -b 600036,600900,601288
```

### 4.2 批量基金

```bash
stock-analyze -b 159934:fund,510310:fund,159928:fund
```

### 4.3 混合类型（股票 + 基金）

```bash
stock-analyze -b 600036:stock,159934:fund,001316:fund
```

> 格式：`代码:类型`，stock 可省略（默认值），fund 必须显式指定。

### 4.4 批量结果保存到文件

```bash
# JSON 格式
stock-analyze -b 600036,600900 --json --pretty -o result.json

# 表格格式
stock-analyze -b 600036,600900 -o result.txt
```

## 五、持仓管理

### 5.1 添加持仓

```bash
# 添加股票
stock-analyze --add 600519

# 添加 ETF 基金
stock-analyze --add 159934:fund

# 添加开放式基金
stock-analyze --add 001316:fund
```

### 5.2 查看持仓列表

```bash
stock-analyze --list
```

输出示例：
```
  持仓列表 (3 项):
      1. 600519
      2. 159934  [基金]
      3. 001316  [基金]

  配置文件: /Users/你的用户名/.stock-analysis/config.json
```

### 5.3 删除持仓

```bash
stock-analyze --remove 600519
```

### 5.4 使用持仓列表批量分析

```bash
# 实时分析
stock-analyze --watchlist

# 离线测试
stock-analyze --watchlist --test

# 保存结果
stock-analyze --watchlist --json --pretty -o watchlist_result.json
```

## 八、Python 代码调用

除了命令行，也可以在 Python 代码中直接调用：

```python
from stock_analysis import StockAnalyzer

analyzer = StockAnalyzer()

# 单只分析
result = analyzer.analyze("600519")
print(result["analysis"]["score"])        # 评分
print(result["analysis"]["trend"])         # 趋势
print(result["analysis"]["recommendation"]) # 建议

# 批量分析
result = analyzer.analyze_batch(["600036", "600900", "159934:fund"])
for r in result["results"]:
    if "analysis" in r:
        si = r["stock_info"]
        an = r["analysis"]
        print(f"{si['code']} {si['name']} 评分:{an['score']} {an['trend']}")

# 离线测试
result = analyzer.analyze_with_mock_data("600519")
```

## 九、常见问题

### Q1: 安装时报错 `pip install` 权限不足

```bash
# macOS/Linux 加 --user
pip install --user -e .

# 或使用虚拟环境
python -m venv .venv
source .venv/bin/activate   # macOS/Linux
.venv\Scripts\activate      # Windows
pip install -e ".[full,dev]"
```

### Q2: 港美股数据获取失败

- yfinance 在国内可能需要代理
- 确认已安装：`pip install yfinance`
- 运行 `stock-analyze --check` 检查数据源连通性

### Q3: 基金代码被识别为股票

00/01 开头的开放式基金与股票代码重叠，必须加 `-t fund`：

```bash
# 错误：被识别为股票
stock-analyze 001316

# 正确：显式指定为基金
stock-analyze 001316 -t fund
```

### Q4: 终端中文显示乱码

```bash
# 使用 ASCII 模式
stock-analyze 600519 --ascii

# 或安装 wcwidth 改善中文对齐
pip install wcwidth
```

### Q5: 数据源全部失败

```bash
# 1. 检查网络连接
ping baidu.com

# 2. 运行环境自检
stock-analyze --check

# 3. 使用离线测试模式
stock-analyze 600519 --test
```

### Q6: 如何更新项目代码

如果使用 `pip install -e .` 开发模式安装，直接 `git pull` 后代码立即生效，无需重新安装。

## 十、配置文件

配置文件自动创建在 `~/.stock-analysis/config.json`，支持自定义默认参数：

```json
{
  "watchlist": ["600519", "159934:fund"],
  "defaults": {
    "market": "auto",
    "days": 60,
    "asset_type": "stock"
  }
}
```

- `watchlist`：自选股列表（通过 `--add` / `--remove` 管理）
- `defaults`：默认参数（未在命令行显式指定时自动读取）

## 十一、Web 可视化界面

除了命令行，项目还提供了基于 Vue 3 + FastAPI 的 Web 可视化界面。

### 11.1 启动 Web 服务

```bash
# 方式一：使用启动脚本（推荐）
cd stock-analysis

# 方式二：手动启动

cd stock-analysis

cd stock-analysis/frontend
```

启动后访问：
- **前端界面**：http://localhost:5173
- **API 文档**（Swagger）：http://localhost:8000/docs
- **API 文档**（ReDoc）：http://localhost:8000/redoc

### 11.2 前端安装

首次使用需要安装前端依赖：

```bash
cd stock-analysis/frontend
npm install
```

### 11.3 页面说明

#### 仪表盘（首页）

- 自选股管理：添加/删除股票，分组筛选
- 一键分析：批量分析所有自选股
- 分析结果：趋势分布饼图 + 评分分布柱状图
- 快速分析：单只股票快速查询

#### 个股分析

- 搜索栏：输入代码 + 选择市场/类型/天数
- K 线图：蜡烛图 + MA 均线叠加 + 成交量
- 技术指标图：MACD / RSI / KDJ / OBV 子图
- 评分面板：综合评分 + 趋势判断 + 操作建议
- 指标详情：MA/MACD/RSI/KDJ/BOLL/Volume/ATR/OBV/CCI/WR
- 关键价位：支撑位 + 压力位

#### 历史记录

- 分析历史列表：支持按代码/趋势/日期筛选
- 评分趋势图：折线图展示评分变化
- 数据导出：导出筛选结果为 CSV

#### 价格预警

- 创建预警：设置价格高于/低于/涨幅/跌幅条件
- 检查预警：一键检查所有未触发预警
- 预警列表：查看状态（待触发/已触发）

### 11.6 技术指标说明

系统支持 9 大维度共 19 条技术指标时序数据：

| 维度 | 指标 | 权重 | 说明 |
|------|------|------|------|
| MA 均线 | MA5/10/20/60 | ±20 | 多头/空头排列判断 |
| MACD | DIF/DEA/MACD | ±15 | 金叉/死叉 + 背离检测 |
| RSI | RSI6/12/24 | ±15 | 超买/超卖判断 |
| KDJ | K/D/J | ±8 | 超买超卖 + 金叉死叉 |
| 布林带 | 上轨/中轨/下轨 | ±10 | 轨道位置 + 开收窄判断 |
| 成交量 | VMA5/VMA10/量比 | ±5 | 放量/缩量判断 |
| OBV | 能量潮 | — | 资金流向趋势 |
| CCI | 顺势指标 | — | 超买(>100)/超卖(<-100) |
| WR | 威廉指标 | — | 超买(>-20)/超卖(<-80) |

### 11.7 评分体系

综合评分 0-100 分，对应 5 个趋势等级：

| 分数区间 | 趋势 | 建议 |
|---------|------|------|
| >= 75 | 强势上涨 | 可适当关注 |
| 60-74 | 上涨趋势 | 可适当关注 |
| 40-59 | 震荡整理 | 建议观望 |
| 25-39 | 下跌趋势 | 建议减仓 |
| < 25 | 强势下跌 | 建议回避 |
