## 3. 已发现的问题和bug

### 3.1 数据库相关问题
- **数据库初始化警告**：系统启动时出现 `(sqlite3.OperationalError) no such table: config` 警告
- **影响**：系统仍能正常运行，但初始化过程不完整

### 3.2 依赖管理问题
- **缺少httpx依赖**：系统运行时出现 `ModuleNotFoundError: No module named 'httpx'` 错误
- **影响**：导致部分功能无法正常运行

### 3.3 API路径问题
- **路径不一致**：测试过程中发现API路径与预期不符
- **影响**：导致测试失败和集成问题

### 3.4 输入验证问题
- **空代码验证**：系统允许空代码提交
- **代码长度验证**：系统允许过长的代码提交
- **影响**：可能导致系统异常或性能问题

### 3.5 错误处理问题
- **错误信息格式不一致**：不同API端点的错误信息格式不统一
- **影响**：前端处理错误时需要适配不同格式

### 3.6 Watchlist API 内部错误
- **问题**：`/watchlist` 端点返回 "服务器内部错误"
- **原因**：`_normalize_watchlist` 函数未过滤空代码和过长代码，导致 Pydantic 验证失败
- **影响**：用户无法获取自选股列表

### 3.7 数据清理问题
- **问题**：配置文件中存在无效的股票代码（空代码或过长代码）
- **影响**：导致API调用失败

### 3.8 前端服务端口变更
- **问题**：前端服务默认端口 5173 被占用，自动切换到 5174 端口
- **影响**：用户需要使用新的端口访问应用
- **建议**：在文档中说明端口可能会自动变更的情况
